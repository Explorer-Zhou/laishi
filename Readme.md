# 来食 - 基于Nodejs的牛马觅食辅助系统

> **⚠️ 警告：** 本项目可能导致深夜饥饿、食欲大增或体重微调，开发者概不负责。

## 🫡 项目简介

"来食"——一款专治**选择困难症晚期患者**、**"随便都行"口头禅持有者**以及**"不知道吃啥"综合征**受害者的救命神器。

是不是每天一到饭点，大脑就自动开始播放《今天吃什么》的死循环？
- 问你的饭搭子，他说："随便。"
- 问你的对象，她说："都行，但不想吃油腻的、不想吃太素的、不想吃昨天吃过的……"

别慌！让 AI 来替你背锅……啊不，替你做主！

系统会根据你的口味偏好（哪怕你是那个吃火锅必点清汤的异教徒）、近期营养摄入情况（不仅是卡路里刺客，还是热量警察），通过 LLM 大模型给你算上一卦，还能顺手把附近的馆子给你捞出来。

**主打一个：甚至比你那句"随便"更懂你自己。**

## 🚀 功能特性

### 🌟 核心科技（真·黑科技）
- 🧠 **赛博算命式点餐**：AI 大脑帮你做决定，从此告别抛硬币和数花瓣。你的下一顿，何必是你自己选的？
- 📊 **营养摄入监控**：精准计算你的卡路里，发现你最近总是"炸鸡一时爽，体重火葬场"。
- 📍 **干饭地图导航**：一键搜索附近美食，拯救你的腿，更拯救你的胃。导航精准度堪比老司机带路。
- 👤 **无痛式账户**：不用手机号，不用验证码，起个网名就能用。主打一个"来去自由，渣男式体验"。
- 🔑 **API 自由主义**：支持自带 Key，想用 GPT 还是其他兼容模型随你挑，我的 AI 我做主。

### 🌭 气氛组功能
- 🍽️ **口味私人订制**：如果你是"香菜党"或者"折耳根绝缘体"，请大声告诉 AI，它不记仇。
- 📈 **干饭大数据**：生成你的专属饮食报表，看看你是"肉食恐龙"还是"吃草达人"。
- 📱 **响应式魔法**：手机、电脑、iPad 都能看，随时随地，想吃就吃，不论是在摸鱼还是在带薪拉屎。
- 🎨 **颜值即正义 UI**：界面好看得让你想多刷即便，哪怕不吃看看也饱了（并不是）。

## 🛠️ 技术栈

### 后端（幕后煮夫）
- **框架**：`Express.js` —— 也就是那个稳如老狗的 Node 框架。
- **数据库**：`SQLite` —— 只有更轻，没有最轻，能跑就行，要什么自行车。
- **LLM服务**：`OpenAI API` —— 调用赛博大脑，负责胡……哦不，负责精准推荐。
- **地图服务**：`高德地图 API` —— 毕竟迷路这种事，还是交给专业的来。
- **语言**：`JavaScript` —— JS 在手，天下我有。

### 前端（门面担当）
- **框架**：`React` + `Vite` —— 前端界的"神雕侠侣"，快到飞起。
- **路由**：`React Router` —— 指哪打哪，永不迷路。
- **API调用**：`Axios` —— 数据搬运工，勤勤恳恳。
- **语言**：`JavaScript` —— 是的，还是它，全栈就是这么任性。

## 📦 安装与运行

### 准备工作（环境检查）
- Node.js >= 18.0.0（太老的版本跑不动，别为难老年人）
- npm >= 9.0.0（装包必备，没它你连大门都进不去）

### 极速上车步骤

1.  **把代码搞下来**
    ```bash
    git clone https://github.com/Explorer-Zhou/laishi.git
    cd laishi
    # 恭喜你，迈出了成为干饭王的第一步
    ```

2.  **给后端喂饭（装依赖）**
    ```bash
    cd backend
    npm install
    # 此时可以去泡杯咖啡，或者刷两个短视频
    ```

3.  **配置神秘代码（环境变量）**
    ```bash
    cp .env.example .env
    # 赶紧打开 .env 文件，填上你的 API Key
    # 别留着 "your-xxx"，那玩意儿跑不起来的
    ```

4.  **给前端喂饭（装依赖）**
    ```bash
    cd ../frontend
    npm install
    # 这一次可以喝口水，很快就好
    ```

### 🚀 启动引擎

1.  **唤醒后端**
    ```bash
    cd backend
    node index.js
    # 看到 "服务器运行在 http://localhost:3001" 时，稳了。
    ```

2.  **唤醒前端**
    ```bash
    cd frontend
    npm run dev
    # 看到 "Local: http://localhost:5173/" 时，起飞！
    ```

3.  **开启干饭之旅**
    打开浏览器访问 `http://localhost:5173`
    > ⚠️ 前方高能，请备好纸巾擦口水。

## 🔌 API 接口概览

> 适合喜欢钻研代码的极客和不想看界面的大佬。

### 用户（甲方爸爸）
- `POST /api/users` - 建个号，从此就是一家人。
- `GET /api/users/:name` - 查户口。
- `PUT /api/users/:name/preferences` - 告诉 AI："我不爱了，换个口味"。

### 营养（健康检测）
- `POST /api/nutrition/record` - 甚至可以记账你偷吃了什么。
- `GET /api/nutrition/analysis/:userName` - 生成一份让你愧疚（或骄傲）的报告。

### 推荐（核心魔法）
- `GET /api/recommendations/food/:userName` - **全场最佳功能**。
- `GET /api/recommendations/nearby` - 哪怕你在荒郊野外……好吧那样可能搜不到。
